# Self-hosted Fonts

Run this script once to download all font files (requires internet access):

```bash
node scripts/download-fonts.js
```

Or download manually from Google Fonts:
- Montserrat: https://fonts.google.com/specimen/Montserrat
  Weights needed: 400 (Regular), 600 (SemiBold), 700 (Bold), 900 (Black)
- Inter: https://fonts.google.com/specimen/Inter
  Weights needed: 400 (Regular), 500 (Medium), 600 (SemiBold)

Save as woff2 files with these exact names:
  montserrat-400.woff2
  montserrat-600.woff2
  montserrat-700.woff2
  montserrat-900.woff2
  inter-400.woff2
  inter-500.woff2
  inter-600.woff2
